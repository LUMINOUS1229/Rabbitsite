# UI 设计方案要点：海风博物馆

> **版本**: 1.0
> **日期**: 2024-07-29

## 1. 设计理念：海风博物馆

**关键词**: 开放、呼吸感、精致、故事感

我们旨在打造一个如同坐落在海边的数字博物馆。访客在此能感受到海风拂面般的清爽与宁静，同时又能细细品味每一件“展品”（作品与研究）背后的深度与故事。设计将强调留白、秩序感和精致的细节，营造一个沉浸式、非干扰的浏览体验。

## 2. 色彩系统

调色板从海边的自然元素中汲取灵感，营造明亮、通透、宁静的氛围。

| 颜色 | HEX | RGB | 用途 | Tailwind 变量 |
| :--- | :--- | :--- | :--- | :--- |
| **海盐白 (Salt White)** | `#F8F8F8` | `248, 248, 248` | 页面主背景色，提供呼吸感。 | `bg-salt-white` |
| **深海蓝 (Deep Sea Blue)** | `#003366` | `0, 51, 102` | 主文本、标题、重要链接、图标。 | `text-deep-sea` |
| **贝壳粉 (Shell Pink)** | `#FCE9E5` | `252, 233, 229` | 点缀色、高亮背景、链接悬停效果。 | `bg-shell-pink` |
| **落日金 (Sunset Gold)** | `#FFD700` | `255, 215, 0` | 行动号召按钮 (CTA)、特殊高光。 | `bg-sunset-gold` |
| **中性灰 (Neutral Gray)** | `#A0A0A0` | `160, 160, 160` | 辅助文本、边框、细线。 | `text-neutral-gray` |

**色彩应用原则**:
- **主导**: 大面积使用海盐白作为背景，创造开阔的空间感。
- **对比**: 深海蓝作为信息承载的主要颜色，确保内容清晰易读。
- **点缀**: 贝壳粉和落日金用于需要吸引用户注意力的元素，如悬停效果和关键按钮，但使用上保持克制，避免喧宾夺主。
- **层次**: 使用不同深浅的灰色来区分信息的次要层级。

## 3. 字体系统

字体选择旨在平衡现代感与可读性，同时兼顾中英文的和谐共存。

- **英文**: **Inter**
  - **特点**: 现代、清晰、中性，屏幕显示效果极佳。
  - **字重**: 
    - `Regular (400)`: 正文内容。
    - `Medium (500)`: 次级标题、加粗文本。
    - `SemiBold (600)`: 主标题。
    - `Bold (700)`: 特别强调的数字或短语。

- **中文**: **思源黑体 (Source Han Sans)**
  - **特点**: 优雅、稳定，与 Inter 搭配和谐。
  - **字重**: 
    - `Regular`: 正文内容。
    - `Medium`: 次级标题、加粗文本。
    - `Bold`: 主标题。

**排版原则**:
- **行高**: 设置为字号的 `1.7` 倍，确保段落呼吸感。
- **字间距**: 适当增加 `letter-spacing`，提升阅读舒适度。
- **段落间距**: 使用足够的 `margin-bottom` 来清晰地分隔段落。

## 4. 组件设计

组件是构成页面的基本单元，其设计需保持一致性、可重用性和精致感。

| 组件 | 设计要点 |
| :--- | :--- |
| **导航栏 (Header)** | - **布局**: 左侧为 Logo/网站标题，右侧为导航链接 (Work, Research, Writing, About, Contact) 和语言切换按钮。<br>- **样式**: 背景透明或为海盐白，滚动时可附着在顶部并带有细线底边。<br>- **交互**: 当前页面链接高亮（深海蓝），鼠标悬停时链接变为贝壳粉。 |
| **按钮 (Button)** | - **主按钮 (CTA)**: 落日金背景，深海蓝文字，圆角，有轻微阴影。<br>- **次要按钮**: 透明背景，深海蓝边框和文字。悬停时背景变为贝壳粉。<br>- **文本链接**: 深海蓝文字，带有下划线。悬停时下划线消失，文字变为贝壳粉。 |
| **卡片 (Card)** | - **用途**: 用于作品集和文章列表。<br>- **样式**: 海盐白背景，圆角，带有非常纤细的灰色边框或轻微阴影。<br>- **布局**: 上方为图片，下方为标题、摘要和标签。<br>- **交互**: 鼠标悬停时，卡片轻微上浮，阴影加深。 |
| **表单 (Form)** | - **输入框**: 细线边框（中性灰），聚焦时边框变为深海蓝。<br>- **标签**: 位于输入框上方，使用深海蓝文字。 |
| **页脚 (Footer)** | - **内容**: 版权信息、社交媒体链接 (GitHub, LinkedIn等)、RSS 订阅链接。<br>- **样式**: 简洁，文字使用中性灰，与页面主体内容有明显区分。 |
| **图标 (Icon)** | - **风格**: 采用线条为主、带有复古感的透明图标库（如 `Feather Icons` 或类似的风格）。<br>- **颜色**: 主要使用深海蓝，交互时可变为贝舍粉。 |

## 5. 动效原则

动效的目的是增强用户体验，而非分散注意力。所有动效应遵循**快速、微妙、有意义**的原则。

- **状态过渡**: 颜色、背景、阴影等属性的变化应使用 `transition`，时长 `150ms` 到 `300ms`。
- **页面切换**: 页面间的切换可以加入轻微的淡入淡出效果，增强流畅感。
- **加载动画**: 对于需要等待加载的内容，使用骨架屏 (Skeleton) 或一个简约的加载指示器（如一个旋转的细圈）。
- **悬停效果**: 交互元素的悬停效果（如卡片上浮、按钮变色）应快速响应，给予用户即时反馈。

## 6. 响应式设计

- **移动端 (Mobile)**: 单栏布局，导航折叠为汉堡菜单，字体大小适中，确保触摸目标足够大。
- **平板端 (Tablet)**: 可根据屏幕宽度采用一或两栏布局。
- **桌面端 (Desktop)**: 充分利用宽屏，采用多栏布局，展示更多信息，留白更充分。
