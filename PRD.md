# 产品需求文档 (PRD)：修修 Xiuqi Li 个人网站

> **版本**: 1.0
> **日期**: 2024-07-29

## 1. 项目概述

### 1.1. 项目目标

为跨领域的创作者“修修 Xiuqi Li”构建一个专业、现代且富有个人特色的中英双语个人网站。该网站旨在：

- **作品展示**：集中展示其在 CS-PhD 研究、艺术创作、UI/UX 设计、AIGC 探索和平面设计等领域的精选项目。
- **研究介绍**：清晰呈现其学术背景、研究方向和成果。
- **文集同步**：整合并展示其个人思考与深度文章。
- **建立连接**：提供明确的联系方式，促进潜在的学术与商业合作。

### 1.2. 目标受众

- **学术界同行**：寻求学术交流与合作的研究者、教授。
- **行业招聘方**：寻找具有交叉学科背景（技术+设计）人才的公司 HR 或团队负责人。
- **潜在客户/合作方**：寻求设计服务、项目合作的个人或组织。
- **普通访客**：对 AIGC、教育游戏、设计等领域感兴趣的关注者。

## 2. 功能需求

### 2.1. 核心功能

| 功能模块 | 描述 | 优先级 |
| :--- | :--- | :--- |
| **中英双语** | 支持用户在中文和英文之间一键切换，内容同步更新。 | 高 |
| **作品集展示 (Work)** | - 作品列表页：以网格或列表形式展示所有项目，含缩略图、标题、标签。<br>- 作品详情页：包含项目背景、过程、成果、方法论、高清图片/视频、外部链接等。 | 高 |
| **研究介绍 (Research)** | - 静态页面，介绍个人研究方向、已发表论文、学术项目等。 | 高 |
| **文章列表 (Writing)** | - 从公众号「Cos兔」同步精选文章，以摘要形式展示。<br>- 支持按分类或标签筛选。 | 中 |
| **关于我 (About)** | - 静态页面，详细介绍个人经历、技能、理念和成就。 | 高 |
| **联系表单 (Contact)** | - 使用 Formspree 提供一个简单、可靠的联系表单，用于接收外部信息。 | 高 |
| **内容管理 (CMS)** | - 使用 Contentlayer 将 MDX 文件作为数据源，方便管理作品、文章等内容。 | 高 |

### 2.2. 非功能性需求

| 需求类别 | 描述 |
| :--- | :--- |
| **技术栈** | - **前端框架**: Next.js 15 (App Router)<br>- **语言**: TypeScript<br>- **样式**: Tailwind CSS<br>- **内容**: MDX + Contentlayer<br>- **部署**: Vercel<br>- **分析**: Plausible<br>- **表单**: Formspree |
| **设计风格** | - **主题**: 「海风博物馆」<br>- **色板**: 海盐白 (#F8F8F8), 深海蓝 (#003366), 贝壳粉 (#FCE9E5), 落日金 (#FFD700)<br>- **字体**: 思源黑体 (中文), Inter (英文)<br>- **布局**: 留白、呼吸感、细线分割<br>- **图标**: 复古、透明感 |
| **性能** | - **加载速度**: 核心页面首次内容绘制 (FCP) < 2.5s。<br>- **移动端优先**: 响应式设计，确保在手机、平板、桌面端均有良好体验。 |
| **可访问性 (A11y)** | - **语义化 HTML**: 使用正确的 HTML 标签。<br>- **对比度**: 文本与背景对比度符合 WCAG AA 标准。<br>- **键盘导航**: 所有交互元素均可使用键盘访问。<br>- **图像 Alt 文本**: 为所有有意义的图片提供描述性 `alt` 属性。 |
| **SEO** | - **元数据**: 为每个页面生成唯一的 `title` 和 `description`。<br>- **OG Image**: 为社交媒体分享自动生成 Open Graph 图片。<br>- **站点地图**: 自动生成 `sitemap.xml`。<br- **RSS**: 为文章提供 RSS 订阅源。 |
| **部署与维护** | - **持续部署**: Git push 自动触发 Vercel 部署。<br>- **源码管理**: 提供完整的源码仓库链接。 |

## 3. 信息架构与路由

网站采用扁平化的信息架构，确保用户能快速触达核心内容。

| 路径 | 页面名称 (英文) | 页面名称 (中文) | 描述 |
| :--- | :--- | :--- | :--- |
| `/` | Home | 首页 | 网站入口，展示个人简介、核心亮点和内容导航。 |
| `/work` | Work | 作品 | 作品集列表页。 |
| `/work/[slug]` | Work Detail | 作品详情 | 单个作品的详细介绍页。`[slug]` 为作品的唯一标识。 |
| `/research` | Research | 研究 | 学术研究介绍页。 |
| `/writing` | Writing | 文集 | 文章列表页。 |
| `/about` | About | 关于 | 个人详细介绍页。 |
| `/contact` | Contact | 联系 | 联系方式与表单页。 |

## 4. 数据模型 (Contentlayer Schema)

### 4.1. `Work` (作品)

```typescript
// content/work/your-project.mdx
{
  title: string,
  description: string,
  date: string, // ISO 8601 格式
  thumbnail: string, // 图片路径
  tags: string[], // e.g., ['UI/UX', 'AIGC']
  language: 'en' | 'zh',
  // MDX 正文内容
}
```

### 4.2. `Writing` (文章)

```typescript
// content/writing/your-post.mdx
{
  title: string,
  summary: string,
  date: string, // ISO 8601 格式
  tags: string[],
  language: 'en' | 'zh',
  // MDX 正文内容
}
```
